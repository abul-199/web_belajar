Cara Mengerjakan Website Belajar Ini

Website ini dibuat supaya kamu belajar web seperti kursus harian. Jadi bukan satu halaman panjang, melainkan dipisah per topik dan per hari agar tidak ribet.

Urutan belajar yang disarankan

  1. Mulai dari Track HTML, kerjakan Hari 1 sampai Hari 30.
  2. Lanjut ke Track CSS supaya struktur HTML yang sudah kamu buat bisa didesain dengan rapi.
  3. Setelah itu masuk ke Track PHP untuk belajar backend.
  4. Terakhir pelajari Track MySQL untuk basis data.

Isi setiap halaman harian

  • Penjelasan materi: konsep dan landasan hari itu.
  • Canvas/editor: tempat mengetik ulang starter code.
  • Referensi solusi: bisa ditampilkan setelah kamu mencoba sendiri.
  • Study case: tugas mini yang harus kamu kerjakan.
  • Pilgan & essay: latihan pemahaman sesuai hari itu.

Cara latihan yang benar

  1. Jangan langsung buka referensi kode.
  2. Ketik dulu dengan tanganmu sendiri di editor.
  3. Lihat preview untuk HTML dan CSS.
  4. Kalau salah, baru bandingkan dengan referensi.
  5. Tambahkan modifikasi kecil supaya benar-benar paham.

Untuk PHP dan MySQL

Pada halaman PHP dan MySQL, editor dipakai untuk latihan sintaks dan logika. Preview visual tidak dijalankan di browser statis. Supaya bisa praktik nyata, jalankan file PHP di server lokal seperti XAMPP/Laragon dan jalankan query MySQL di phpMyAdmin atau terminal database.

Inti sistem ini

Belajar sedikit demi sedikit, mengetik banyak, dan tidak loncat. Itu yang membuat dasar web development jadi kuat.
