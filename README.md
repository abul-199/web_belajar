# WebMaster Academy

Project website belajar web multi-page dengan sistem harian.

## Struktur
- index.html -> dashboard
- html.html / css.html / php.html / mysql.html -> halaman track
- pages/<topic>/day-01.html s.d day-30.html -> materi harian
- assets/ -> CSS, JS, dan data materi
- php-crud-example/ -> contoh project PHP + MySQL untuk latihan server lokal

## Cara pakai
1. Extract ZIP.
2. Buka index.html di browser.
3. Pilih track.
4. Klik kartu Hari 1 - Hari 30.
5. Kerjakan editor, study case, quiz, dan essay.
